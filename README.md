# foxGo2
## It is a GO program and 2nd generation of foxGo, which is written by PyQt5, with independent GO engine. The first purpose of this application is viewing all go game on Tencent weiqi easily, next will realize more function. Playing with jueyi is concerned.

### Dependency:
* python >= 3.6
* PyQt5 (python and Qt5 bindings)

### Run:
* navigate to source dir in console and type: `python foxGo2.py` in console to run the application
* or directly click `foxGo2.py` to run
